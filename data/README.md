# Data Dictionary

This folder contains datasets used in **R for Medical Data Analysis**.

## `diabetes`

- **Source package:** `medicaldata`
- **Dataset:** `medicaldata::diabetes`
- **Rows x Columns:** 768 x 9
- **Generated by:** `R/generate_diabetes_data.R`
- **Export files:**
  - `data/diabetes.csv`
  - `data/diabetes.xlsx` (sheet: `diabetes`)

### Regeneration

From project root, run:

```bash
Rscript R/generate_diabetes_data.R
```

### Columns

| Column | Type | Description |
|---|---|---|
| `pregnancy_num` | numeric | Number of pregnancies |
| `glucose_mg-dl` | numeric | Plasma glucose concentration (mg/dL) |
| `dbp_mm-hg` | numeric | Diastolic blood pressure (mmHg) |
| `triceps_mm` | numeric | Triceps skinfold thickness (mm) |
| `insulin_microiu-ml` | numeric | 2-hour serum insulin (microIU/mL) |
| `bmi` | numeric | Body mass index (kg/m^2) |
| `pedigree` | numeric | Diabetes pedigree function score |
| `age` | numeric | Age (years) |
| `diabetes_5y` | factor | Diabetes status at 5 years (`neg`, `pos`) |

### Notes

- Missing values are preserved from the source dataset.
- Column names are kept identical to the source package for traceability.
- If your installed CRAN version of `medicaldata` does not include `diabetes`, this script automatically installs the development version from GitHub (`higgi13425/medicaldata`).

## `heart_disease`

- **Source:** UCI Machine Learning Repository — Heart Disease (Cleveland subset)
- **URL:** <https://archive.ics.uci.edu/ml/machine-learning-databases/heart-disease/processed.cleveland.data>
- **Rows x Columns:** 303 x 14
- **Generated by:** `R/generate_heart_disease_data.R`
- **Export files:**
  - `data/heart_disease.csv`
  - `data/heart_disease.xlsx` (sheet: `data`)

### Regeneration

From project root, run:

```bash
Rscript R/generate_heart_disease_data.R
```

### Columns

| Column | Type | Description |
|---|---|---|
| `age` | integer | Age (years) |
| `sex` | factor | Sex (`female`, `male`) |
| `cp_type` | factor | Chest pain type (`typical_angina`, `atypical_angina`, `non_anginal`, `asymptomatic`) |
| `resting_bp` | integer | Resting blood pressure (mmHg) |
| `chol` | integer | Serum cholesterol (mg/dL) |
| `fbs` | factor | Fasting blood sugar > 120 mg/dL (`no`, `yes`) |
| `resting_ecg` | factor | Resting ECG result (`normal`, `st_t_abnormality`, `lv_hypertrophy`) |
| `max_hr` | integer | Maximum heart rate achieved |
| `exer_ang` | factor | Exercise-induced angina (`no`, `yes`) |
| `st_dep` | double | ST depression induced by exercise relative to rest |
| `st_slope` | factor | Slope of peak exercise ST segment (`upsloping`, `flat`, `downsloping`) |
| `fluoro_vv` | integer | Number of major vessels colored by fluoroscopy (0–3) |
| `thal` | factor | Thalassemia (`normal`, `fixed_defect`, `reversible_defect`) |
| `hd_dx` | integer | Heart disease severity (0 = none, 1–4 = increasing severity) |

### Notes

- **Source encoding:** Original UCI attributes use numeric codes. The generation script converts categorical variables to labeled factors (see column table above).
- **Missing values:** The UCI data uses `?` for missing values (6 total, in `fluoro_vv` and `thal`). These are converted to `NA`.
- **Target variable:** `hd_dx` is kept as integer 0–4 to preserve granularity. Value 0 means no significant narrowing; values 1–4 indicate presence of heart disease with increasing severity.
