# Data Dictionary

This folder contains datasets used in **R for Medical Data Analysis**.

## `diabetes`

- **Source package:** `medicaldata`
- **Dataset:** `medicaldata::diabetes`
- **Rows x Columns:** 768 x 9
- **Generated by:** `R/generate_diabetes_data.R`
- **Export files:**
  - `data/diabetes.csv`
  - `data/diabetes.xlsx` (sheet: `diabetes`)

### Regeneration

From project root, run:

```bash
Rscript R/generate_diabetes_data.R
```

### Columns

| Column | Type | Description |
|---|---|---|
| `pregnancy_num` | numeric | Number of pregnancies |
| `glucose_mg-dl` | numeric | Plasma glucose concentration (mg/dL) |
| `dbp_mm-hg` | numeric | Diastolic blood pressure (mmHg) |
| `triceps_mm` | numeric | Triceps skinfold thickness (mm) |
| `insulin_microiu-ml` | numeric | 2-hour serum insulin (microIU/mL) |
| `bmi` | numeric | Body mass index (kg/m^2) |
| `pedigree` | numeric | Diabetes pedigree function score |
| `age` | numeric | Age (years) |
| `diabetes_5y` | factor | Diabetes status at 5 years (`neg`, `pos`) |

### Notes

- Missing values are preserved from the source dataset.
- Column names are kept identical to the source package for traceability.
- If your installed CRAN version of `medicaldata` does not include `diabetes`, this script automatically installs the development version from GitHub (`higgi13425/medicaldata`).
